# Roshen KSA — Forecast & Scenario Lab

أداة توقّع خفيفة (ملف HTML واحد، تشتغل في المتصفح بدون سيرفر) تقرأ الرو داتا من
إكسل وتصدّر التوقّع إلى إكسل.

## الاستخدام
1. افتح `forecast-tool.html` في المتصفح (دبل-كليك).
2. اسحب ملف الرو داتا (نفس أعمدة التقارير: `Item Id`, `Invoice Amount ex Vat`,
   `Inv Qty Cases`, `Branch`, `Region2`, `Item Category`, `IsReturn`, `Day/Month/Year`…).
   الأعمدة تُكتشف تلقائياً.
3. اضبط الافتراضات (نافذة الأساس، الأفق، معامِلات الموسم) وأضف لِيفرات السيناريو
   (بروموشن / ضغط صنف).
4. صدّر إكسل متعدّد الشيتات: Forecast (SKU×Branch) · By Category · By Branch ·
   Scenario Summary · Assumptions · Backtest.

## المنهجية
- المحرّك على مستوى **الصنف × الفرع** ويتجمّع لأي مستوى (صنف/فئة/فرع/منطقة/إجمالي).
- الأساس = متوسط متحرك موزون لآخر N شهر مكتمل، × معامِلات موسم يدوية،
  مع معالجة الشهر الجاري الناقص بـ run-rate.
- المبيعات **بدون ضريبة** (`Invoice Amount ex Vat`) وصافي بعد المرتجعات.
- **Backtest**: يخبّي آخر شهر مكتمل ويقيس WMAPE والانحياز — مؤشّر صدق للدقّة.

## ملاحظة عن الموسمية
الداتا الحالية سنة واحدة وفيها ذروة رمضان/العيد (فبراير–مارس) ثم هبوط بعد العيد.
لذلك معامِلات الموسم اليدوية ضرورية — استخدم زر "بريست رمضان/العيد" كنقطة بداية وعدّله.
